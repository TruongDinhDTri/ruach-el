## 🗂️ How to Use This

Each file = one concept, one mental model, one "aha moment."

When you learn something new:
1. Create a new `.md` file in the right category folder
2. Write it in YOUR words — not copy-paste from docs
3. Add a real example from the bot's codebase
4. Link related concepts together

## 📝 Note Template

When creating a new note, use this format:

```markdown
# [Concept Name]

**What it is:** One sentence definition in YOUR words.

**Why it matters for the bot:** How this concept shows up in our actual code.

**The mental model:** The metaphor or image that made it click.

**Real example:** Actual code snippet or decision from the bot.

**Related concepts:** Links to other notes.
```
