```

├── 📜 rauch-el-update_memory.py   # The Evolution Engine
│
├── 📂 prompts/                     # The Soul (Immutable)
│   ├── system.txt                 # Core identity & mission
│   ├── persona.txt                # Personality & style
│   ├── mission.txt                # Purpose & objectives
│   ├── behavior.txt               # Interaction patterns
│   └── end-session.txt            # Closing ritual
│
├── 📂 memory/                      # The Mind (Evolving)
│   ├── global-memory.md           # Core identity & long-term facts
│   ├── relationship-memory.md     # Interaction style & preferences
│   └── project-memory.md          # Template for new projects
│
└── [PROJECT FOLDER]/               # Per-Project Context
    └── project-context.md          # Active project state (God Source)
```

### Memory Classification

| Memory File | Type | Update Frequency | Purpose |
|-------------|------|------------------|---------|
| `global-memory.md` | IMMUTABLE | Rare | Fundamental truths about Wiganz |
| `relationship-memory.md` | FLUID | Moderate | How Ruach-El interacts with Wiganz |
| `project-context.md` | ACTIVE | Per session | Current project decisions, tasks |

### The God Source Principle

The `project-context.md` in each project folder is the **God Source of Truth**:
- **READ** at session start (to compile context)
- **WRITTEN** at session end (via update_memory.py)
- **PROTECTED** from accidental overwrite
- Contains: decisions, tasks, The Thread, growth notes
